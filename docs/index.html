Vibeshed

A keyboard-driven macOS launcher built with SwiftUI

Control your Mac with keystrokes. Manage windows, switch apps, search browser tabs, control Spotify, and more from a single floating picker.

View on GitHub Install

Features

✨

Smart Launcher

Fuzzy search with usage-aware sorting and context-sensitive action boosting. Adapts to how you work.

▣

Window Management

Tile, resize, cycle sizes, maximize/restore. Anchor to edges, split halves, focus by title.

🌐

Browser Control

Search and switch tabs across Safari and Chrome. Bookmarks, history, per-URL routing to browser profiles.

📋

Clipboard History

Persistent clipboard history with search, paste, and regex-based exclusion patterns.

🎵

Spotify & Audio

Search and play music. Control volume, mute, select audio devices, media keys.

🛠

Developer Tools

VSCode, JetBrains, iTerm, GitHub, and AI session search. Open projects, run commands, browse PRs.

⌨

Key Remapping

Global and per-app key combos. Remap keys, bind mouse buttons, use Caps Lock as hyper key.

🎨

Dynamic Theming

Theme shifts with your system appearance and the vibe of what you're doing.

Get Started

Requires macOS 14 (Sonoma) or later and Swift 5.9+.

git clone https://github.com/idmitriev/vibeshed.git
cd vibeshed
make build
make run

The app lives in your menu bar. Configure it at ~/.config/vibeshed/config.yaml. See config.example.yaml for all options.
